# Photo-gallery App :fox_face:
## Store your photos.

## Features-
* Upload photos
* dark mode 
* Firebase storage
* Pwa app(progressive web app)-React app can be installed in various devices and used
* Local storage for themes
* Delete photos
* Signin through google
* Only Registered users can delete/upload photos
